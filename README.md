## О проекте

**PasswordForge** создан для тех, кто ценит безопасность и не хочет запоминать сложные комбинации.  
Основная цель – предоставить *простой интерфейс* для генерации паролей, соответствующих современным стандартам безопасности.

> «Слабые пароли – открытая дверь для злоумышленников. PasswordForge закрывает её за вас» — из технического задания[^1].

Каждый сгенерированный пароль гарантированно содержит буквы, цифры и специальные символы, что делает его устойчивым к атакам перебора.  
Программа написана на **Python** с использованием только стандартной библиотеки (`random`, `string`, `argparse`).

[^1]: Внутренняя документация проекта, раздел «Требования к надёжности», версия 2.1.

## Возможности

- ✅ Генерация паролей заданной длины (от 6 до 128 символов)
- ✅ Настраиваемые наборы символов:
  - строчные латинские буквы (`a-z`)
  - заглавные латинские буквы (`A-Z`)
  - цифры (`0-9`)
  - специальные символы (`!@#$%^&*`)
- ✅ Исключение неоднозначных символов (`l`, `1`, `O`, `0`)
- ✅ Кроссплатформенность (Windows, Linux, macOS)
- ✅ Возможность генерации нескольких паролей за один запуск
